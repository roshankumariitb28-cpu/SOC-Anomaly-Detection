## Video Inference

The trained YOLOv5 model was tested on sample videos using the `detect.py` script. The model successfully processed each frame, detected the objects, and generated output videos with bounding boxes. The output videos were saved in the `runs/detect/` directory and verified locally.

**Note:** One sample inference video has been uploaded to the GitHub repository. The remaining output videos were not uploaded because GitHub's web interface has a **25 MB file upload limit**, and the generated videos exceed this limit.
